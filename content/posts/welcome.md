Hello and welcome to the TacticalBlog! OK, that's a stupid name, so maybe not. It's all a WIP.

Anyway, you've made it to my new website. This is the long awaited HTML version of my Arknights Mastery Guide. The Google Sheet is great, but can sometimes be slow and buggy, and some people can't use it at all. It's long been an item on my backlog of projects, and now I'm finally getting to it. Plus, now I have a place to write about my random musings that aren't worth posting as Reddit content for those rare few who might actually read it.

So that said, a few house keeping topics.

First, no the Sheet version will not be going anywhere. If that is your preferred way to consume my content, that will remain the same and will for the foreseeable future. When the Gamepress site shit the bed, I got to experience first hand the problems that come with moving things around. A lot of people only occasionally come to the guide, and I have no intention of leaving them in the dust! The content between this version and the Sheets version *should* be identical. And if you spot a difference, please let me know so I can fix it!

Along the same lines, I will continue to publish the articles to Lungmen Dragons, Sanity;Gone, and Reddit. The more the merrier. My intention has *never* been about traffic or anything like that. Just helping! So the more people it reaches, the better. Although I admit, it is more work for me maintaining all these versions... Alas, I do it for you, dear reader.

Last, this website is an evolving work in progress. While I have a software background, I've never done web development, let alone modern development. If you spot things that are broken, or unexpected, or even if you have suggestions, please let me know. I'm very receptive to feedback as I figure this all out. That said, frameworks like Hugo and tools like Claude make this all very easy! Man things sure have changed since I first got into software way back when...
